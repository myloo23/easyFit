# Architecture Decision Records

Use this directory for major architectural decisions.

## Naming

`ADR-XXXX-short-title.md`

Example:

`ADR-0001-repository-architecture.md`

## Template

```md
# ADR-XXXX: Title

Status:
Date:

## Context

## Decision

## Alternatives Considered

## Benefits

## Trade-offs

## Privacy Impact

## Security Impact

## Scalability Impact

## Reversal Strategy
```

Do not create ADRs just to fill the directory. Create them when a meaningful decision is being made.
